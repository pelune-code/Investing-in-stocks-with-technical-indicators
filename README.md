# Investing in Stocks with Technical Indicators

This repository contains a Jupyter Notebook project focused on comparison
of performance of technical indicators. The project is packaged as a ZIP 
file containing Jupyter notebook, thanks to which the bachelor's thesis 
"Investing in stock price with technical indicators" was created.

---

## Project Overview

The notebook walks through:
- Loading historical stock data
- Computing statistics of stock prices
- Testing technical indicators
- Visualizing trends and signals
- Simulating own stock prices data to demonstrate functionality of technical indicators
